blog
--------

[Description]

Install:

    npm install blog
	
Add dependency to your project with

    "blog": "git@github.com/Michieljoris/blog"
	
or

	"blog": "*"

Require in a module:

    var blog = require('blog');

Use:

See [documentation](https://rawgithub.com/Michieljoris/blog/master/docs/blog.html).
